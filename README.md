***ИНСТРУКЦИЯ ПО ПРОГРАММЕ АЗС "ЗАПРАВКА Мичурики"***

**Откройте файл "Uch_Prak.exe". Появится главное меню.**

Введите цифру и нажмите Enter.

*ПУНКТ 1: ЦЕНЫ*

Показывает таблицу с видами топлива, ценами и типами.


*ПУНКТ 2: ЗАПРАВИТЬ*


Шаг 1: Введите номер авто (например, А123БВ)

Шаг 2: Если вы новый клиент:

Введите имя

Введите начальный баланс (сколько денег кладете)

Шаг 3: Введите ID топлива из таблицы (1-5)

Шаг 4: Выберите номер колонки

Шаг 5: Введите количество литров

Шаг 6: Если денег хватает - заправка выполнена


*ПУНКТ 3: СТАТИСТИКА*

Показывает общую выручку, количество клиентов и остатки в колонках.


*ПУНКТ 4: ПОПОЛНИТЬ БАЛАНС*

Введите номер авто

Введите сумму пополнения

Баланс обновится


*ПУНКТ 5: ИСТОРИЯ*

Введите номер авто

Показывает все прошлые заправки: дата, топливо, литры, стоимость


*ПУНКТ 6: ПОПОЛНИТЬ КОЛОНКУ (для работников)*

Показывает остатки во всех колонках

Выберите номер колонки

Введите количество литров для долива


*ПУНКТ 7: ВЫХОД*

Закрывает программу.


**ВАЖНО**

Номер авто можно вводить любыми буквами (а123бв = А123БВ)

Если денег на счету мало - заправка не пройдет

Если в колонке мало бензина - программа предложит залить сколько есть

При ошибках ввода программа вернется в меню
